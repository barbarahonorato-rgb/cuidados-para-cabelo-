Dicas de ouro

  • Lave o cabelo com agua morna,nunca quente.
  • Evite usar secador e chapinha em excesso.
    • Use mascara de hidrataçao pelo menos uma vez por semana. Faça cortes regulares para evitar pontas dupla.
  • Aplique protetor termico antes de usar ferramentas de calor.
  • Use óleos finalizadores para selar as pontas.
  • Faça umectação semanal com óleos vegetais.
